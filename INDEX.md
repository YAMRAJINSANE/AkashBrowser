Check Node.js installed: `node --version`\n2. Reinstall dependencies: `rm -rf node_modules && npm install`\n3. Check React server: Wait 10 seconds for http://localhost:3000\n4. Open DevTools: F12, check for errors\n\n### Issue: Chrome not found\n**Solution**:\n1. Install Google Chrome\n2. Verify path: `C:\\Program Files\\Google\\Chrome\\Application\\chrome.exe`\n3. Set CHROME_PATH environment variable\n\n### Issue: Port 3000 in use\n**Solution**:\n```bash\n# Windows\nnetstat -ano | findstr :3000\ntaskkill /PID <PID> /F\n\n# macOS/Linux\nlsof -i :3000\nkill -9 <PID>\n```\n\n### More Help\n- See [SETUP_GUIDE.md](./SETUP_GUIDE.md) Troubleshooting section\n- Check `%APPDATA%\\Akash Browser\\logs\\` for error logs\n- Open DevTools: F12\n\n---\n\n## 📊 Technology Stack\n\n| Component | Technology | Version |\n|-----------|-----------|----------|\n| Desktop | Electron | 26.0.0 |\n| Frontend | React | 18.2.0 |\n| Styling | Tailwind CSS | 3.3.3 |\n| Animations | Framer Motion | 10.16.4 |\n| Icons | Lucide | 0.263.1 |\n| Database | SQLite | 5.1.6 |\n| Build | Electron Builder | 24.5.0 |\n\n---\n\n## ✨ Key Highlights\n\n### Professional Design\n- Modern UI with dark mode\n- Smooth animations\n- Glass morphism effects\n- Responsive layout\n\n### Robust Architecture\n- Secure IPC communication\n- Process isolation\n- Error handling\n- Logging system\n\n### Production Ready\n- Complete documentation\n- Build configuration\n- Security best practices\n- Performance optimization\n\n### Developer Friendly\n- Hot reload\n- DevTools integration\n- Clear code structure\n- Comprehensive comments\n\n---\n\n## 🎓 Learning Resources\n\n### Official Documentation\n- [Electron Docs](https://www.electronjs.org/docs)\n- [React Docs](https://react.dev)\n- [Tailwind CSS Docs](https://tailwindcss.com/docs)\n- [Framer Motion Docs](https://www.framer.com/motion/)\n\n### Tutorials in This Project\n- IPC Communication -
